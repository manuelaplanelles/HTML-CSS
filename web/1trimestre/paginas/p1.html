Volver arriba

¿Qué parte del código HTML está un poco obsoleto en el código del ejercicio 7?

El atributo name utilizado en las etiquetas <a name="..."> para crear anclas o marcadores está obsoleto en HTML5. Aunque todavía funciona por compatibilidad con navegadores antiguos, la forma moderna y recomendada es utilizar el atributo id directamente en cualquier elemento HTML. Por ejemplo, en lugar de <a name="primero"></a>, se debería usar <h1 id="primero">Primer apartado</h1>. Esta práctica es más semántica, limpia el código eliminando etiquetas vacías innecesarias y es el estándar actual según las especificaciones del W3C para HTML5.

Además, el atributo bgcolor en la etiqueta <body> también está obsoleto en HTML5. La forma moderna es usar CSS, ya sea con estilos en línea style="background-color: #c7aa92;" o preferiblemente en un archivo CSS externo.
